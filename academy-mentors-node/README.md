# REST API in Node.js 

**Technologies used**

* Node.js
* express.js
* sqllite

**Available routes**

`GET /api/articles` — get all articles
`GET /api/article/:id` — get an article by id
`POST /api/article` — create an article
`PUT /api/article/:id` — edit an article
`DELETE /api/article/:id` — delete an article by id
